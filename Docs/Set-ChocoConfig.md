# Set-ChocoConfig

## SYNOPSIS
Sets the specified configuration option.

## SYNTAX

```
Set-ChocoConfig [-Name] <String> [-Value] <String> [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
Sets the specified configuration option.

## EXAMPLES

### EXAMPLE 1
```
Set-ChocoConfig -Name containsLegacyPackageInstalls -Value true
```

# Get-ChocoPackageInfo

## SYNOPSIS
Gets information about a Chocolatey package.

## SYNTAX

```
Get-ChocoPackageInfo [-Name] <String> [<CommonParameters>]
```

## DESCRIPTION
Gets information about a Chocolatey package.

## EXAMPLES

### EXAMPLE 1
```
Get-ChocoPackageInfo -Name rufus
```
